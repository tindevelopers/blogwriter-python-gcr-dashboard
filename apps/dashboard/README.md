# Dashboard App

The main dashboard application for the BlogWriter Python GCR project.

## Development

```bash
# From the root of the monorepo
npm run dev

# Or run just the dashboard
cd apps/dashboard
npm run dev
```

## Features

- Provider management
- Configuration settings
- Monitoring and analytics
- Real-time updates with Firebase

## Tech Stack

- Next.js 16
- React 19
- Firebase
- TanStack Query
- Recharts for data visualization
